A BMO handheld device with a joystick and an OLED screen. The device features four options to choose from:

    BMO static face
    Snake game
    Brick destroyer
    Drawing

Components Used

    ESP32 30-pin WROOM module
    1.5-inch OLED screen
    PSP joystick module
    1 push button
    TP4056 charging module
    803550 LiPo battery
    1 push button switch
